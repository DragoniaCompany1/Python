Apa itu Python?

Bayangkan jika Anda bisa berbicara dengan komputer menggunakan bahasa yang hampir mirip bahasa Inggris sehari-hari. Itulah Python!

Python adalah bahasa pemrograman yang diciptakan oleh Guido van Rossum pada tahun 1991. Dia ingin membuat bahasa yang:

  • Mudah dibaca seperti membaca koran
  • Mudah ditulis seperti menulis pesan WhatsApp
  • Powerful seperti superkomputer

Instalasi Python - Step by Step

Untuk Windows (Paling Mudah)

Step 1: Download

  1. Buka browser, ketik: python.org
  2. Klik tombol kuning besar "Download Python"
  3. File akan download otomatis

Step 2: Install

  1. Klik dua kali file yang di-download
  2. PENTING : âœ… Centang "Add Python to PATH"
  3. Klik "Install Now"
  4. Tunggu sampai selesai (3-5 menit)

Step 3: Test Instalasi

  1. Tekan Windows + R
  2. Ketik cmd , tekan Enter
  3. Ketik: python --version
  4. Kalau muncul "Python 3.x.x" berarti berhasil! ğŸ‰

Untuk Mac

Cara Termudah - Pakai Homebrew:

  1. Buka Terminal (Cmd + Space, ketik "Terminal")
  2. Copy-paste ini: 
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  1. Setelah selesai, ketik:
